# 🎬 MovieFlix — Movie Recommendation Web App

A Netflix-inspired movie recommendation web app built with Python & Streamlit.

## 🔗 Live Demo
👉 [Click here to try MovieFlix](https://movieflix-app-kr3rw7tlk3mrggebk7pwrc.streamlit.app)

## 📌 Features
- 🔍 Smart movie recommendations using **Cosine Similarity**
- 🎭 Filter by **Genre, Mood & Year**
- ❤️ Personal **Watchlist** management
- 📊 **Analytics Dashboard** with Top 10 rated movies
- 🎬 3,296 movies with posters, ratings & genres

## 🛠️ Tech Stack
| Technology | Purpose |
|------------|---------|
| Python | Core language |
| Pandas | Data processing |
| Scikit-learn | Recommendation algorithm |
| Streamlit | Web app & deployment |
| GitHub | Version control |

## 🚀 How to Run Locally
```bash
git clone https://github.com/Reshma620/movieflix-app.git
cd movieflix-app
pip install -r requirements.txt
streamlit run app.py
```
